# TEENZ 리포트 시스템 — 시작 가이드

## 1단계: Node.js 설치 (최초 1회)

### 방법 A — 공식 설치 파일 (가장 쉬움)
1. https://nodejs.org 접속
2. **LTS** 버튼 클릭 → 다운로드
3. 설치 파일 실행 (계속 "다음" 클릭)
4. 터미널 새로 열기

### 방법 B — Homebrew 이용
터미널에서 아래 명령어 실행:
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
이후:
```
brew install node
```

---

## 2단계: 프로젝트 의존성 설치

터미널에서 아래 명령어를 **차례대로** 입력:

```bash
# 폴더로 이동
cd ~/Desktop/teenz-report

# 백엔드 패키지 설치
npm install

# 프론트엔드 패키지 설치
cd client
npm install
cd ..

# 샘플 데이터 생성
npm run setup
```

---

## 3단계: 실행

```bash
cd ~/Desktop/teenz-report
npm run dev
```

브라우저에서 아래 주소로 접속:
- **관리자 화면**: http://localhost:5173
- **학부모 화면 샘플**: 서버 실행 후 터미널에 출력된 링크 확인

---

## 매일 사용할 때

터미널 열고:
```bash
cd ~/Desktop/teenz-report
npm run dev
```

끝낼 때: 터미널에서 `Ctrl + C`

---

## 프로덕션 배포 (나중에)

```bash
cd ~/Desktop/teenz-report
npm run build     # 프론트엔드 빌드
node server.js    # 서버만 실행 (포트 3001)
```
